# Round Robin Web

## How to Use
Clone the GitHub Repo:

```bash
git clone https://github.com/Juliancpp/roundRobin_Web.git
```
Acces the folder:
```bash
cd roundRobin_Web/
```

### Install dependencies

You can use one of them `npm`, `yarn`, `pnpm`, `bun`, Example using `npm`:

```bash
npm install
```

### Run the development server

```bash
npm run dev
```
